One main function of an operating system is security. OS security is the process of protecting the operating system's confidentiality. It is in charge of protecting from viruses, worms, or any other malware so it does not slow your computer down. It is also in charge of preventing hackers or any other unautherized users from having access to your data.
Another fuction of the operating system is memory management. Memory management is refered to as the internal memory of a computer. This memory is also known as RAM and it's where every program is loaded to for execution. Therefore, every program or file we access is copied into the main memory.
File managemnet is the collection of information stored in a compuers memory. It helps manage new files and storing them in specific locations making it easier to locate them in your computer files. These files are information that is stored in what is called secondary storage.
